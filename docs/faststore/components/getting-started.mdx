---
title: "Getting started with FastStore UI"
---

FastStore UI is an ecommerce-focused library of [brandless](https://developers.vtex.com/docs/guides/faststore/using-themes-overview#brandless-and-the-theming-structure) components you can easily use on your application.
It provides features & basic style that makes it faster to implement and customize on top of it.

## Installation

Install `@faststore/ui` as a dependency of your project via the command line:

### yarn

```sh
yarn add @faststore/ui
```

### npm

```sh
npm install @faststore/ui
```

---

## Adding styles

Import one of the following options:

### All components included

```sh
@import "@faststore/ui/src/styles/components.scss";
```

### Base styles 

```sh
@import "@faststore/ui/src/styles/base.scss";
```
You should also import the desired components' style individually by checking the component references to import its style.
